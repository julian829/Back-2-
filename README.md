# Proyecto Astrologic

Sistema básico de eventos meteorológicos desarrollado con Spring Boot.

## Estructura del proyecto

- Modelo
- Repositorio
- DTO
- Mapper
- Clase de prueba

## Uso de IA en el proyecto

Durante el desarrollo del proyecto se realizaron algunas correcciones con ayuda de herramientas de inteligencia artificial.

Por ejemplo:
- Se mejoró la organización del DTO, ya que inicialmente no teníamos claro cómo estructurarlo correctamente.
- Se ajustó la conversión entre el modelo y el DTO mediante el uso del mapper.
- Se corrigieron pequeños detalles de estructura del proyecto y paquetes.

El archivo README también fue ajustado con apoyo de IA para documentar mejor el proyecto.
